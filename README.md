# PIC16F690 LCD Digital Clock

Jednoduché digitální hodiny napsané v ASM pro PIC16F690 s LCD 2x16 (HD44780, 4-bit mód).

## Features
- Zobrazení času: HH:MM:SS
- Nastavení času pomocí tlačítek
- ASCII reprezentace času
- Podpora českých znaků

## Hardware
- MCU: PIC16F690
- LCD: HD44780 (4-bit)
- Tlačítka: 3x (H, M, SET)
- Clock: 4 MHz

## Build
MPASM (Microchip)

## Notes
Starší low-level projekt, žádné knihovny, čistý ASM.
